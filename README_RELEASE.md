# AppJoos Desktop Agent v0.3.0

## What it does
AppJoos Desktop Agent is a local desktop monitoring tool that provides:

- System health monitoring
- Application usage tracking
- Usage insights and charts
- Passive network monitoring
- Network insights
- User-triggered active internet speed tests
- Tray/background operation

## Main executable
Folder build:
- dist/AppJoosDesktopAgent/AppJoosDesktopAgent.exe

## Notes
- Speed tests only run when the user clicks Run Speed Test
- Passive network monitoring is separate from active speed testing
- UI polish/performance refinements are planned for the next version
